# QR Code mesa

## O que e

O QR Code mesa permite que o cliente faca pedidos diretamente da mesa usando o proprio celular.

## Para que serve

Ele reduz o tempo de espera e amplia a autonomia do cliente durante o atendimento.

## Quando utilizar

Use este canal em restaurantes, bares e cafeterias.

## Como funciona na operacao

O cliente escaneia o QR Code, escolhe os produtos e envia o pedido.

Depois disso, o pedido segue para a Central de Pedidos.
