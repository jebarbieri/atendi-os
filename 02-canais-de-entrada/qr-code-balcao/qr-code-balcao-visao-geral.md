# QR Code balcao

## O que e

O QR Code balcao permite pedidos rapidos pelo celular em operacoes de balcao.

## Para que serve

Ele ajuda a diminuir filas e aumentar a velocidade do atendimento.

## Quando utilizar

Use este canal em operacoes de balcao.

## Como funciona na operacao

O cliente acessa o cardapio digital pelo QR Code e envia o pedido.

Depois disso, o pedido segue para a Central de Pedidos.
