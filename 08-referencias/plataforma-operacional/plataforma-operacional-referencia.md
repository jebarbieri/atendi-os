# Plataforma operacional

## O que significa

Na visao da Atendi, uma plataforma operacional e um sistema que conecta os processos centrais da loja em vez de cuidar apenas de uma funcao isolada.

## Diferenca para ferramentas pontuais

Enquanto solucoes pontuais resolvem partes do problema, uma plataforma operacional busca coordenar vendas, atendimento, pedidos, gestao, dados e rotinas da operacao.

## Como isso muda a leitura do produto

Em vez de perguntar apenas quais funcionalidades o sistema tem, a documentacao atual sugere olhar para como a loja funciona melhor quando tudo conversa.

## Relacao com o Atendi OS

A Atendi nao se posiciona apenas como PDV tradicional, ERP, sistema de vendas ou sistema de delivery. A proposta atual e de plataforma operacional para lojas modernas.
