# Unidades operacionais

## O que sao

As unidades operacionais sao os setores responsaveis pela execucao dos pedidos.

## Para que servem

Elas organizam quem executa cada parte do fluxo dentro da operacao.

## Exemplos citados na base atual

- cozinha
- bar
- separacao
- oficina
- servicos
- expedicao

## Como funciona na operacao

Cada pedido pode ser enviado automaticamente para uma ou mais unidades operacionais.
