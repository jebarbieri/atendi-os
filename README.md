# Atendi OS

Base publica de documentacao do produto Atendi OS.

## Objetivo

Centralizar a documentacao oficial do produto em um repositorio proprio, separado da base interna da Atendi.

## Como navegar

- `produto-visao-geral.md`: entrada principal da base
- `SUMMARY.md`: navegacao completa da documentacao
- `.gitbook.yaml`: configuracao pronta para publicacao no GitBook
